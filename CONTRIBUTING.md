# Contributing to Agent Pilot

Thank you for contributing to Agent Pilot.

## License of Contributions

By submitting a pull request, patch, issue comment with code, or any other
contribution intended for inclusion in this repository, you agree that your
contribution is licensed under the same license as the project: Apache-2.0.

Unless you explicitly state otherwise, your contribution may be used,
modified, and redistributed as part of Agent Pilot under the Apache License,
Version 2.0.

## Copyright and Attribution

Please do not submit code, images, text, or other material unless you have the
right to contribute it.

Where practical, new source files should include SPDX headers:

```text
SPDX-FileCopyrightText: 2026 Sheldon Van
SPDX-License-Identifier: Apache-2.0
```

Third-party code or assets should keep their original copyright notices and
license terms.
